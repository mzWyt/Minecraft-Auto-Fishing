# Minecraft-Auto-Fishing
A script for auto fishing in Minecraft using template matching.

### 用法：运行run.py / run.exe

---------------------------------------

1. 进入Minecraft后选择字幕出现范围检测范围
2. 点击“启动”
3. 向水中抛出鱼竿
